## How to Run the Application

1. Clone the repository:

   ```sh
   git clone https://github.com/quangthao2002/Food_FS.git
   cd food-be
   ```

2. Install the dependencies:

   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run server
   ```
